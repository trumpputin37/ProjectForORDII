# ProjectForORDII

Анализ экологических данных по регионам Российской Федерации за 2014 год. Включает загрузку, очистку, визуализацию и статистический анализ данных о загрязнении воздуха, качестве воды и выбросах.

data_environment - исходные данные

project.py - основной скрипт анализа

README.md - документация
